## Component Specifications

### ProjectCard
- Thumbnail image (16:9 ratio)
- Title (truncate at 60 chars)
- Author avatar + name
- Tags (max 3 visible)
- Status badge (color coded)
- Failure log count
- Hover: subtle scale(1.02) + shadow

### FailureLogCard
- Collapsible by default (show goal + outcome summary)
- Expand to show full structured log
- Mentor review indicator (if exists)
- Timestamp
- Link to full log

### MentorBadge
- Verified checkmark icon (muted cyan)
- Tooltip: "Platform-verified mentor"
- Community verified: additional star icon

### TokenBadge
- Diamond icon + count
- Tooltip: "Tokens earned through contributions"

### Timeline
- Vertical line with nodes
- Different node types: project start, failure log, iteration, endorsement
- Color coding by type
- Hover to preview content
